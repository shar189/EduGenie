# EduGenie - AI Powered Educational Learning Assistant

EduGenie is an AI-powered educational learning assistant designed to help students learn more effectively through intelligent educational support.

## Features

- Question Answering
- Concept Explanation
- Quiz Generation
- Text Summarization
- Personalized Learning Path
